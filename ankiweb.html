This add-on allows to delay/postpone updates for selected add-ons. This might be useful if you have made some custom modifications to installed add-ons and you want to delay an update which would overwrite these modifications. This might be useful if you're curious what an update changed because some add-ons only have minimal documentation so that sometimes looking at the source code is informative. Maybe it's also useful if you worry about security, see e.g. this discussion. # Configuration There are some options in the config that are disabled by default. You can e.g. enable to automatically diff the addons folder before and after the update. For this the add-on can copy the addons folder to a temp folder before updating. This feature is untested and since it optionally deletes file be careful before using it. # Compatibility This add-on overwrites two functions from Anki - namely "handle_update_info" and "prompt_to_update" from aqt.addons. # Alternatives There's also the add-on Addons update picker by epiphanie-gedeon. Both add-ons serve a similar purpose and were made by accident at the same time. # bugs Use this version at your own risk. This add-on is newly released and virtually no one has downloaded it so far.
