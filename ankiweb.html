This add-on allows to delay/postpone updates for selected add-ons. For some motivation see e.g. this discussion. # Compatibility This add-on overwrites two functions from Anki - namely "handle_update_info" and "prompt_to_update" from aqt.addons. # bugs Use this version at your own risk.
